# Política de Privacidade

Última atualização: 23 de Abril de 2026

1. Introdução

O LayerZeppelin.pt ("nós", "nosso" ou "site") está comprometido em proteger a sua privacidade. Esta Política de Privacidade explica como recolhemos, usamos, armazenamos e protegemos os seus dados pessoais em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD/GDPR) - Regulamento (UE) 2016/679.

2. Responsável pelo Tratamento

LayerZeppelin.pt

Email: contact@layerzeppelin.pt

Este site é um portefólio pessoal com fins de demonstração de competências em cibersegurança.

3. Dados Pessoais que Recolhemos

Podemos recolher os seguintes tipos de dados pessoais:

  • Dados de navegação: endereço IP, tipo de navegador, dispositivo, páginas visitadas, tempo de permanência (anónimos e agregados)
  • Cookies: conforme descrito na nossa Política de Cookies
  • Dados de contacto: caso envie email para contact@layerzeppelin.pt

4. Finalidade do Tratamento

Os dados são tratados para as seguintes finalidades:

  • Funcionamento técnico do site
  • Análise estatística anónima de visitas
  • Resposta a pedidos de contacto
  • Cumprimento de obrigações legais

5. Base Legal para o Tratamento

De acordo com o RGPD, o tratamento de dados baseia-se em:

  • Consentimento (Art. 6.º, n.º 1, al. a) - para cookies não essenciais
  • Interesse legítimo (Art. 6.º, n.º 1, al. f) - para analytics anónimos
  • Execução de contrato (Art. 6.º, n.º 1, al. b) - quando contactar por email

6. Partilha de Dados

NÃO partilhamos os seus dados pessoais com terceiros para fins de marketing.

Os dados podem ser partilhados com:

  • Prestadores de serviços técnicos (alojamento web, CDN) - sob contrato de confidencialidade
  • Autoridades competentes, se exigido por lei

7. Serviços de Terceiros

Este site utiliza:

  • Cloudflare Pages: alojamento e CDN (pode processar IPs de forma temporária)
  • Cloudflare Web Analytics: estatísticas anónimas (não utiliza cookies)

8. Cookies

Consulte a nossa Política de Cookies para informação detalhada sobre os cookies utilizados.

9. Os Seus Direitos (RGPD)

Nos termos do RGPD, tem os seguintes direitos relativamente aos seus dados pessoais:

  • Direito de acesso (Art. 15.º): obter confirmação e cópia dos seus dados
  • Direito de retificação (Art. 16.º): corrigir dados inexatos ou incompletos
  • Direito de apagamento (Art. 17.º): solicitar a eliminação dos seus dados
  • Direito à limitação (Art. 18.º): restringir o processamento
  • Direito à portabilidade (Art. 20.º): receber dados em formato estruturado
  • Direito de oposição (Art. 21.º): opor-se ao processamento
  • Direito de retirar consentimento: a qualquer momento, sem afetar a licitude do tratamento anterior

Para exercer os seus direitos, contacte-nos em: contact@layerzeppelin.pt

10. Período de Conservação

Os dados pessoais são conservados apenas pelo tempo necessário para:

  • Fornecer os serviços solicitados
  • Cumprir obrigações legais
  • Resolver disputas e fazer cumprir acordos

Dados de contacto: conservados durante 3 anos após o último contacto, salvo solicitação de eliminação.

11. Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger os seus dados:

  • Conexão HTTPS (TLS 1.2/1.3)
  • Cabeçalhos de segurança HTTP (CSP, X-Frame-Options, HSTS)
  • Scripts de terceiros não utilizados
  • Sem tracking de terceiros

12. Transferências Internacionais

Os dados podem ser processados pela Cloudflare (EUA), que utiliza Cláusulas Contratuais Padrão (SCC) aprovadas pela UE para garantir proteção adequada em conformidade com o RGPD.

13. Direito de Reclamação

Se considerar que o processamento dos seus dados pessoais viola o RGPD, tem o direito de apresentar uma reclamação junto da autoridade de controlo competente:

CNPD - Comissão Nacional de Proteção de Dados

Av. D. Carlos I, 134 - 1.º, 1200-651 Lisboa, Portugal

Website: www.cnpd.pt

14. Alterações à Política de Privacidade

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas através de aviso destacado no site.

15. Contacto

Para qualquer questão sobre privacidade:

Email: contact@layerzeppelin.pt
